The user can directly convert into other units by input to converters like Length, Volume, Mass


steps:
1. Input any value in the field to convert
2. The conversions are produced
    - Meters to Feet and Feet to Meters (Length)
    - Liters to Gallons and Gallons to Liters (Volume)
    - Kilograms to Pounds and Pounds to Kilograms (Mass)
